Introduction to BlackRoad OS

BlackRoad OS is a distributed infrastructure platform for deploying and managing edge workers, AI agents, and serverless applications at scale.

New to BlackRoad?
Start with the Quick Start guide to deploy your first worker in under 5 minutes.

What is BlackRoad OS?

BlackRoad OS provides a unified control plane for managing distributed infrastructure across multiple cloud providers, edge networks, and on-premise systems. It combines the simplicity of serverless with the power of container orchestration.

Key Features

  • Edge Workers — Deploy JavaScript/TypeScript functions to 300+ edge locations worldwide
  • AI Agents — Orchestrate autonomous agents with built-in memory and coordination
  • Global Storage — Distributed KV, SQL, and object storage with automatic replication
  • Zero Config Networking — Automatic DNS, TLS, and routing configuration

Quick Installation

Install the BlackRoad CLI using npm:

bash 
# Install globally
npm install -g @blackroad/cli

# Verify installation
blackroad --version

# Login to your account
blackroad login

Create Your First Worker

Initialize a new project and deploy it to the edge:

typescript 
// src/index.ts
export default {
  async fetch(request: Request): Promise<Response> {
    return new Response("Hello from BlackRoad!", {
      headers: { "content-type": "text/plain" }
    });
  }
};
Requires Node.js 18+
Make sure you have Node.js version 18 or higher installed before proceeding.

Configuration Options

Option Type Default Description
name string — The name of your worker
main string src/index.ts Entry point for your worker
routes array [] Custom domain routes
vars object {} Environment variables
